#AngularJS boilerplate 

Includes:
* ui-router
* sass
* gulp

Fork or clone, then run npm install.


##Folder structure:

1. ###dist
- all.css
- all.js

2. ###js
- controllers
    - mainCtrl.js
- directives
    - mainDirective.js
    - navbar.js
- services
    - mainService.js
- app.js

3. ###styles
- base
    - reset.css
- fonts
    - fonts.css
- views
    - main.css
    - main.scss

4. ###views
- home.html
- navbar.html
- random.html

5. gulpfile.js
6. index.js
7. package.json
8. readme.md
